详细描述

template<typename T>
class cudf::detail::pinned_allocator< T >

pinned_allocator is a CUDA-specific host memory allocator that employs cudaMallocHost for allocation.

This implementation is ported from the experimental/pinned_allocator that Thrust used to provide.

参见
https://en.cppreference.com/w/cpp/memory/allocator

This method allocates storage for objects in pinned host memory.

参数
cntThe number of objects to allocate.
返回
a pointer to the newly allocated objects.
注解
The second parameter to this function is meant as a hint pointer to a nearby memory location, but is not used by this allocator.
This method does not invoke value_type's constructor. It is the responsibility of the caller to initialize the objects at the returned pointer.

参数

注解
