Functor used to single-\/type-\/dispatch binary operators. 

This functor\textquotesingle{}s {\ttfamily operator()} is templated to validate calls to its operators based on the input type, as determined by the {\ttfamily is\+\_\+valid\+\_\+binary\+\_\+op} trait. This function assumes that both inputs are the same type, and dispatches based on the type of the left input.
