\doxysubsection{详细描述}
Request for groupby aggregation(s) to perform on a column. 

The group membership of each {\ttfamily value\mbox{[}i\mbox{]}} is determined by the corresponding row {\ttfamily i} in the original order of {\ttfamily keys} used to construct the {\ttfamily groupby}. I.\+e., for each {\ttfamily aggregation}, {\ttfamily values\mbox{[}i\mbox{]}} is aggregated with all other {\ttfamily values\mbox{[}j\mbox{]}} where rows {\ttfamily i} and {\ttfamily j} in {\ttfamily keys} are equivalent.

{\ttfamily values.\+size()} column must equal {\ttfamily keys.\+num\+\_\+rows()}.
