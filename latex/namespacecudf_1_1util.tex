Return the absolute value of a number. 

This calls {\ttfamily std\+::abs()} which performs equivalent\+: {\ttfamily (value \texorpdfstring{$<$}{<} 0) ? -\/value \+: value}.

This was created to prevent compile errors calling {\ttfamily std\+::abs()} with unsigned integers. An example compile error appears as follows\+:
