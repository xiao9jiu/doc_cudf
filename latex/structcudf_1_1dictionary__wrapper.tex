\subsubsection*{template$<$typename Index\+Type$>$\newline
struct cudf\+::dictionary\+\_\+wrapper$<$ Index\+Type $>$}
A strongly typed wrapper for indices in a DICTIONARY type column. 

Index\+Type will be integer types like int32\+\_\+t.

For example, {\ttfamily dictionary32} is a strongly typed wrapper around an {\ttfamily int32\+\_\+t} value that holds the offset into the dictionary keys for a specific element.

This wrapper provides common conversion and comparison operations for the Index\+Type.
