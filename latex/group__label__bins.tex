Labels elements based on membership in the specified bins. 

A bin {\ttfamily i} is defined by {\ttfamily left\+\_\+edges\mbox{[}i\mbox{]}, right\+\_\+edges\mbox{[}i\mbox{]}}. Whether the edges are inclusive or not is determined by {\ttfamily left\+\_\+inclusive} and {\ttfamily right\+\_\+inclusive}, respectively.

A value {\ttfamily input\mbox{[}j\mbox{]}} belongs to bin {\ttfamily i} if {\ttfamily value\mbox{[}j\mbox{]}} is contained in the range {\ttfamily left\+\_\+edges\mbox{[}i\mbox{]}, right\+\_\+edges\mbox{[}i\mbox{]}} (with the specified inclusiveness) and {\ttfamily label\mbox{[}j\mbox{]} == i}. If {\ttfamily input\mbox{[}j\mbox{]}} does not belong to any bin, then {\ttfamily label\mbox{[}j\mbox{]}} is NULL.

Notes\+:
\begin{DoxyItemize}
\item If an empty set of edges is provided, all elements in {\ttfamily input} are labeled NULL.
\item NULL elements in {\ttfamily input} belong to no bin and their corresponding label is NULL.
\item NaN elements in {\ttfamily input} belong to no bin and their corresponding label is NULL.
\item Bins must be provided in monotonically increasing order, otherwise behavior is undefined.
\item If two or more bins overlap, behavior is undefined.
\end{DoxyItemize}
