Operator functor. 

This functor is templated on an {\ttfamily ast\+\_\+operator}, with each template specialization defining a callable {\ttfamily operator()} that executes the operation. The functor specialization also has a member {\ttfamily arity} defining the number of operands that are accepted by the call to {\ttfamily operator()}. The {\ttfamily operator()} is templated on the types of its inputs (e.\+g. {\ttfamily typename LHS} and {\ttfamily typename RHS} for a binary operator). Trailing return types are defined as {\ttfamily decltype(result)} where {\ttfamily result} is the returned value. The trailing return types allow SFINAE to only consider template instantiations for valid combinations of types. This, in turn, allows the operator functors to be used with traits like {\ttfamily is\+\_\+valid\+\_\+binary\+\_\+op} that rely on {\ttfamily std\+::is\+\_\+invocable} and related features.
