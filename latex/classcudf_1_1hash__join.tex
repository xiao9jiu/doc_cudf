\doxysubsection{详细描述}
Hash join that builds hash table in creation and probes results in subsequent {\ttfamily \texorpdfstring{$\ast$}{*}\+\_\+join} member functions. 

This class enables the hash join scheme that builds hash table once, and probes as many times as needed (possibly in parallel).
