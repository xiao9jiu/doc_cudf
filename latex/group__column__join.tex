Returns the exact number of matches (rows) when performing a mixed left semi join between the specified tables where the columns of the equality table are equal and the predicate evaluates to true on the conditional tables. 

If the provided predicate returns NULL for a pair of rows (left, right), that pair is not included in the output. It is the user\textquotesingle{}s responsibility to choose a suitable compare\+\_\+nulls value AND use appropriate null-\/safe operators in the expression.
