\doxysubsection{详细描述}
Abstraction for window boundary sizes, to be used with {\ttfamily \doxylink{group__aggregation__rolling_ga595d27d939a1af9d00d2b6bea3b5623c}{grouped\+\_\+range\+\_\+rolling\+\_\+window()}}. 

Similar to {\ttfamily \doxylink{structcudf_1_1window__bounds}{window\+\_\+bounds}} in {\ttfamily \doxylink{group__aggregation__rolling_ga0d431c50f9d34e8bf6beda13f693bd69}{grouped\+\_\+rolling\+\_\+window()}}, {\ttfamily \doxylink{structcudf_1_1range__window__bounds}{range\+\_\+window\+\_\+bounds}} represents window boundaries for use with {\ttfamily \doxylink{group__aggregation__rolling_ga595d27d939a1af9d00d2b6bea3b5623c}{grouped\+\_\+range\+\_\+rolling\+\_\+window()}}. A window may be specified as either of the following\+:
\begin{DoxyEnumerate}
\item A fixed-\/width numeric scalar value. E.\+g. a) A {\ttfamily DURATION\+\_\+\+DAYS} scalar, for use with a {\ttfamily TIMESTAMP\+\_\+\+DAYS} orderby column b) An {\ttfamily INT32} scalar, for use with an {\ttfamily INT32} orderby column
\item \"{}unbounded\"{}, indicating that the bounds stretch to the first/last row in the group. 
\end{DoxyEnumerate}
