\doxysubsection{详细描述}
Holds column names and buffers containing raw file-\/level and stripe-\/level statistics. 

The buffers can be parsed using a Protobuf parser. Alternatively, use {\ttfamily \doxylink{structcudf_1_1io_1_1parsed__orc__statistics}{parsed\+\_\+orc\+\_\+statistics}} to get the statistics parsed into a libcudf representation.

The {\ttfamily column\+\_\+names} and {\ttfamily file\+\_\+stats} members contain one element per column. The {\ttfamily stripes\+\_\+stats} contains one element per stripe, where each element contains column statistics for each column.
