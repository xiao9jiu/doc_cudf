\doxysubsection{详细描述}
Helper class for computing sort-\/based groupby 

This class serves the purpose of sorting the keys and values and provides building blocks for aggregations. It can provide\+:
\begin{DoxyEnumerate}
\item On-\/demand grouping or sorting of a value column based on {\ttfamily keys} which is provided at construction
\item Group offsets\+: starting offsets of all groups in sorted key table
\item Group valid sizes\+: The number of valid values in each group in a sorted value column 
\end{DoxyEnumerate}

Computes and stores unsorted labels on first invocation and returns stored labels on subsequent calls.
