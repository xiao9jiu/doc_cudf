Adds or subtracts a number of months from the datetime type and returns a timestamp column that is of the same type as the input {\ttfamily timestamps} column. 

For a given row, if the {\ttfamily timestamps} or the {\ttfamily months} column value is null, the output for that row is null. This method preserves the input time and the day where applicable. The date is rounded down to the last day of the month for that year, if the new day is invalid for that month.
