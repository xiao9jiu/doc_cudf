Class definitions for {\ttfamily (mutable)\+\_\+table\+\_\+view} 

A {\ttfamily (mutable\+\_\+)table\+\_\+view} is a set of {\ttfamily (mutable\+\_\+)column\+\_\+view}s of equal size.

A {\ttfamily (mutable\+\_\+)table\+\_\+view} is non-\/owning and trivially copyable and should be passed by value.
