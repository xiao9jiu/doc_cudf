Create a lists column of distinct elements found only in the left input column. 

Given two input lists columns {\ttfamily lhs} and {\ttfamily rhs}, an output lists column is created in a way such that each of its row {\ttfamily i} contains a list of distinct elements that can be found in {\ttfamily lhs\mbox{[}i\mbox{]}} but are not found in {\ttfamily rhs\mbox{[}i\mbox{]}}.

The order of distinct elements in the output rows is unspecified.

A null input row in any of the input lists columns will result in a null output row.
