\doxysubsection{详细描述}
Allows associating a message string with an NVTX event via its {\ttfamily Event\+Attribute}s. 

Associating a {\ttfamily message} with an NVTX event through its {\ttfamily \doxylink{classnvtx3_1_1event__attributes}{event\+\_\+attributes}} allows for naming events to easily differentiate them from other events.

Every time an NVTX event is created with an associated {\ttfamily message}, the contents of the message string must be copied. This may cause non-\/trivial overhead in highly performance sensitive sections of code. Use of a {\ttfamily \doxylink{classnvtx3_1_1registered__message}{nvtx3\+::registered\+\_\+message}} is recommended in these situations.
