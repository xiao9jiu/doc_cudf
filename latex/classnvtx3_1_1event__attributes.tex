\doxysubsection{详细描述}
Describes the attributes of a NVTX event. 

NVTX events can be customized via four \"{}attributes\"{}\+:


\begin{DoxyItemize}
\item color\+: color used to visualize the event in tools such as Nsight Systems. See {\ttfamily color}.
\item message\+: Custom message string. See {\ttfamily message}.
\item payload\+: User-\/defined numerical value. See {\ttfamily payload}.
\item category\+: Intra-\/domain grouping. See {\ttfamily category}.
\end{DoxyItemize}

These component attributes are specified via an {\ttfamily \doxylink{classnvtx3_1_1event__attributes}{event\+\_\+attributes}} object. See {\ttfamily \doxylink{classnvtx3_1_1color}{nvtx3\+::color}}, {\ttfamily \doxylink{classnvtx3_1_1message}{nvtx3\+::message}}, {\ttfamily \doxylink{classnvtx3_1_1payload}{nvtx3\+::payload}}, and {\ttfamily \doxylink{classnvtx3_1_1category}{nvtx3\+::category}} for how these individual attributes are constructed.

While it is possible to specify all four attributes, it is common to want to only specify a subset of attributes and use default values for the others. For convenience, {\ttfamily \doxylink{classnvtx3_1_1event__attributes}{event\+\_\+attributes}} can be constructed from any number of attribute components in any order.
