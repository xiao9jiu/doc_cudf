The result(s) of a \doxylink{group__column__copy_gab171e877b57253d521eb8f6d3c7d7c4a}{cudf\+::contiguous\+\_\+split} 

Each \doxylink{classcudf_1_1table__view}{table\+\_\+view} resulting from a split operation performed by contiguous\+\_\+split, will be returned wrapped in a {\ttfamily \doxylink{structcudf_1_1packed__table}{packed\+\_\+table}}. The \doxylink{classcudf_1_1table__view}{table\+\_\+view} and internal column\+\_\+views in this struct are not owned by a top level \doxylink{classcudf_1_1table}{cudf\+::table} or \doxylink{classcudf_1_1column}{cudf\+::column}. The backing memory and metadata is instead owned by the {\ttfamily data} field and is in one contiguous block.

The user is responsible for assuring that the {\ttfamily table} or any derived table\+\_\+views do not outlive the memory owned by {\ttfamily data}
