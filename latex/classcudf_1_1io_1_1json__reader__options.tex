\doxysubsection{详细描述}
Input arguments to the {\ttfamily read\+\_\+json} interface. 

Available parameters are closely patterned after PANDAS\textquotesingle{} {\ttfamily read\+\_\+json} API. Not all parameters are supported. If the matching PANDAS\textquotesingle{} parameter has a default value of {\ttfamily None}, then a default value of {\ttfamily -\/1} or {\ttfamily 0} may be used as the equivalent.
