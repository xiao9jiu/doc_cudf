Create a lists column in which each row contains a sequence of values specified by a tuple of ({\ttfamily start}, {\ttfamily size}) parameters. 

Create a lists column in which each row is a sequence of values starting from a {\ttfamily start} value, incrementing by one, and its cardinality is specified by a {\ttfamily size} value. The {\ttfamily start} and {\ttfamily size} values used to generate each list is taken from the corresponding row of the input {\ttfamily starts} and {\ttfamily sizes} columns.


\begin{DoxyItemize}
\item {\ttfamily sizes} must be a column of integer types.
\item All the input columns must not have nulls.
\item If any row of the {\ttfamily sizes} column contains negative value, the output is undefined.
\end{DoxyItemize}

Create a lists column in which each row contains a sequence of values specified by a tuple of ({\ttfamily start}, {\ttfamily step}, {\ttfamily size}) parameters. 

Create a lists column in which each row is a sequence of values starting from a {\ttfamily start} value, incrementing by a {\ttfamily step} value, and its cardinality is specified by a {\ttfamily size} value. The values {\ttfamily start}, {\ttfamily step}, and {\ttfamily size} used to generate each list is taken from the corresponding row of the input {\ttfamily starts}, {\ttfamily steps}, and {\ttfamily sizes} columns.


\begin{DoxyItemize}
\item {\ttfamily sizes} must be a column of integer types.
\item {\ttfamily starts} and {\ttfamily steps} columns must have the same type.
\item All the input columns must not have nulls.
\item If any row of the {\ttfamily sizes} column contains negative value, the output is undefined.
\end{DoxyItemize}
