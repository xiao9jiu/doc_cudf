\doxysubsection{详细描述}
Holds column names and parsed file-\/level and stripe-\/level statistics. 

The {\ttfamily column\+\_\+names} and {\ttfamily file\+\_\+stats} members contain one element per column. The {\ttfamily stripes\+\_\+stats} member contains one element per stripe, where each element contains column statistics for each column.
