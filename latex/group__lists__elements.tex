Returns a numeric column containing the number of rows in each list element in the given lists column. 

The output column will have the same number of rows as the input lists column. Each {\ttfamily output\mbox{[}i\mbox{]}} will be {\ttfamily input\mbox{[}i\mbox{]}.size()}.
