\doxysubsection{详细描述}
A wrapper class for operations on a dictionary column. 

A dictionary column contains a set of keys and a column of indices. The keys are a sorted set of unique values for the column. The indices represent the corresponding positions of each element\textquotesingle{}s value in the keys.
