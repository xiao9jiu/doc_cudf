\doxysubsection{详细描述}
Object for intra-\/domain grouping of NVTX events. 

A {\ttfamily category} is simply an integer id that allows for fine-\/grain grouping of NVTX events. For example, one might use separate categories for IO, memory allocation, compute, etc.

Example\+:
