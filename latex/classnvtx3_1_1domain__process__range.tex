\subsubsection*{template$<$typename D = domain\+::global$>$\newline
class nvtx3\+::domain\+\_\+process\+\_\+range$<$ D $>$}
A RAII object for creating a NVTX range within a domain that can be created and destroyed on different threads. 

When constructed, begins a NVTX range in the specified domain. Upon destruction, ends the NVTX range.

Similar to {\ttfamily \doxylink{classnvtx3_1_1domain__thread__range}{nvtx3\+::domain\+\_\+thread\+\_\+range}}, the only difference being that {\ttfamily \doxylink{classnvtx3_1_1domain__process__range}{domain\+\_\+process\+\_\+range}} can start and end on different threads.

Use of {\ttfamily \doxylink{classnvtx3_1_1domain__thread__range}{nvtx3\+::domain\+\_\+thread\+\_\+range}} should be preferred unless one needs the ability to start and end a range on different threads.

{\ttfamily \doxylink{classnvtx3_1_1domain__process__range}{domain\+\_\+process\+\_\+range}} is moveable, but not copyable.
